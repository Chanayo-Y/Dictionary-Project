About Us

We are 3rd year BSIT students who built this portfolio as part of our midterm project. This project challenged us to create a functional web application that interacts with a public API. Our chosen project is the Dictionary Application, which allows users to search for word meanings using a live dictionary API.

Our goal is to showcase our skills in HTML, CSS, and JavaScript — focusing on API integration, teamwork, and user-friendly design.

Why We Chose the Dictionary App

As students, we often encounter unfamiliar words in technical documentation and daily readings. This inspired us to build an application that promotes learning and accessibility through a simple yet powerful dictionary tool.

We chose to integrate the Dictionary API because it allows real-time word definitions, synonyms, and antonyms — making it a valuable resource for students, writers, and developers. Our goal is to bridge technology with education by providing a lightweight and interactive learning tool.

By developing this project, we strengthened our understanding of API requests, JSON data handling, and responsive web design, which are essential skills for modern web developers.
